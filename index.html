Математические выражения

  • N! - факториал числа N
  • loga^N - логарифм чила N по основанию a
  • ?aN - корень a степени из числа N
  • N**a - число N в степени a
  • Ne+a = N*10**a

Горячие клавиши

  • Enter - вычислить
  • Ctrl + Z - на действие назад
  • Ctrl + Shift + Z / Ctrl + Y - на действие вперед
  • Space - фильтрация
  • Q - подсветка ошибок
